# Carlos Nieto - QA Engineer Portfolio

Personal portfolio built with Next.js 14, TypeScript, Tailwind CSS and Framer Motion.

## 🚀 Technologies
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## 🌍 Features
- Responsive design
- Multi-language support (ES/CA/EN)
- Smooth animations
- SEO optimized
